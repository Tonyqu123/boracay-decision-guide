# Boracay Diving Judgment

这个文档定义长滩岛潜水判断框架。

潜水不是普通娱乐商品。

它涉及安全、教练、装备、天气、海况、沟通和用户身体状态，因此判断必须比普通活动更保守。

---

## 1. 活动类型

### 1.1 体验潜水

常见用户：

- 第一次潜水。
- 不确定自己是否喜欢潜水。
- 不会游泳但想尝试。
- 想拍照、打卡、体验海底。

核心判断：

- 教练是否有足够时间做安全讲解。
- 是否小比例带客。
- 是否明确说明不会游泳的限制和风险。
- 是否有泳池或浅水适应环节。
- 是否强行承诺“不会游泳也完全没问题”。

适合：

- 身体健康、愿意听从教练、能接受不确定体验的人。
- 不追求深度和复杂路线，只想安全体验的人。

不适合：

- 恐水严重的人。
- 耳压无法平衡的人。
- 对安全讲解不耐烦的人。
- 只为了拍照但不愿意配合训练的人。

### 1.2 浮潜

常见用户：

- 亲子。
- 不想背气瓶的人。
- 预算有限或时间有限的人。
- 对潜水有顾虑的人。

核心判断：

- 是否提供救生衣。
- 是否有工作人员看护。
- 海况是否适合新手。
- 是否存在强推拍照、强制消费。
- 接送和等待时间是否清晰。

适合：

- 想低门槛接触海水活动的人。
- 带孩子或同行人能力差异大的人。

不适合：

- 对水质、鱼群、珊瑚期待过高的人。
- 不愿意接受天气和海况影响的人。

### 1.3 考证

常见用户：

- 想系统学习潜水的人。
- 未来还想在其他地方 fun dive 的人。
- 愿意投入 2-4 天的人。

核心判断：

- 教练资质和语言是否匹配。
- 课程是否压缩过度。
- 理论、平静水域、开放水域训练是否完整。
- 是否为了低价牺牲教学时间。
- 是否明确说明不过关时如何处理。

适合：

- 时间充足、愿意认真学习的人。
- 能接受学习压力和考试要求的人。

不适合：

- 行程很赶的人。
- 只想轻松玩水、不想学习理论的人。
- 身体状态不稳定或耳压问题明显的人。

### 1.4 Fun Dive

常见用户：

- 已持证潜水员。
- 想看具体潜点的人。
- 对装备、潜导、船程、海况更敏感的人。

核心判断：

- 潜店是否核验证书和近期潜水记录。
- 潜导是否根据经验分组。
- 装备维护是否稳定。
- 潜点选择是否受天气、流、能见度影响。
- 是否有清晰的取消和改期机制。

适合：

- 有证、有经验、能自我管理基础风险的人。

不适合：

- 很久没潜但不愿意做复习的人。
- 只看价格、不关注潜导和装备的人。

## 2. 高风险信号

以下信号需要降低置信度，甚至建议用户避开：

- 安全讲解很短或被用户反复投诉。
- 一个教练同时带太多人。
- 装备老旧、漏气、面镜进水等问题被多次提到。
- 用户提到恐慌后工作人员不耐心。
- 取消、退款、改期规则模糊。
- 天气不佳仍强推下水。
- 不确认身体状况和病史。
- 对不会游泳用户过度承诺。
- 拍照销售压过安全流程。
- 低价套餐隐藏额外费用。

关键问题：

> 这个差评是体验不佳，还是安全管理问题？

安全管理问题必须高权重处理。

## 3. 适配人群判断

### 3.1 不会游泳

不能简单回答“可以”或“不可以”。

需要判断：

- 是否恐水。
- 是否能听懂教练指令。
- 是否能接受面部入水。
- 是否能在浅水区适应。
- 潜店是否有新手流程。
- 是否有一对一或小比例带客。

输出应该更像：

> 不会游泳不一定排除体验潜水，但恐水、无法配合指令、耳压困难的人不适合。需要优先选择安全讲解充分、小比例带客、评价中反复提到耐心的新手友好潜店。

### 3.2 亲子

需要确认：

- 年龄限制。
- 身高和体重限制。
- 家长是否可以陪同。
- 是否有儿童装备。
- 是否更适合浮潜而不是体验潜水。
- 是否有等待区和接送安排。

### 3.3 情侣 / 拍照用户

需要避免把拍照体验置于安全之上。

需要确认：

- 拍照是否包含。
- 是否额外收费。
- 是否影响潜水流程。
- 是否有不拍照的更低成本选择。

### 3.4 认真学习用户

考证用户要更关注教学质量，而不是低价和出片。

需要确认：

- 课程天数。
- 教练语言。
- 理论学习方式。
- 训练是否完整。
- 学员教练比例。
- 是否支持补课或不过关处理。

## 4. 信息源判断

### 4.1 平台商品页

商品页适合看：

- 活动包含什么。
- 接送范围。
- 取消政策。
- 时长。
- 是否含拍照。
- 年龄和健康限制。

但商品页通常不会充分暴露：

- 教练耐心。
- 装备维护。
- 实际等待时间。
- 海况变化。
- 安全讲解质量。

### 4.2 Google Maps / TripAdvisor

适合看：

- 潜店长期口碑。
- 教练名字是否被反复正向提及。
- 差评是否集中在安全、沟通、退款。
- 国际游客对专业性的评价。

### 4.3 中文平台 / 小红书

适合看：

- 中文用户是否能顺利沟通。
- 不会游泳、亲子、新手体验。
- 拍照、接送、等待时间。
- 是否存在被推销或预期落差。

风险：

- 种草内容可能淡化安全细节。
- 拍照好看不等于潜水流程可靠。
- 单篇爆款体验不能代表稳定履约。

---

## 5. 下单前必须确认

体验潜水：

- 教练和学员比例是多少？
- 是否有中文或用户能听懂的安全讲解？
- 不会游泳是否可以参加，具体限制是什么？
- 是否有浅水适应环节？
- 拍照是否包含，是否强制？
- 天气不好如何取消或改期？

浮潜：

- 是否提供救生衣？
- 是否有人看护？
- 浮潜点在哪里？
- 是否适合儿童和不会游泳的人？
- 接送和等待时间大概多久？

考证：

- 课程几天？
- 语言是什么？
- 教练资质如何确认？
- 学员教练比例是多少？
- 如果无法完成课程如何处理？
- 证书费用是否包含？

Fun dive：

- 是否核验证书？
- 是否需要 refresh？
- 潜点如何决定？
- 装备是否包含？
- 取消和改期规则是什么？
- 是否按经验分组？

## 7. 判断底线

潜水判断要避免输出：

> 不会游泳也没问题，放心冲。

更好的输出是：

> 不会游泳不一定不能体验潜水，但这不是无风险活动。当前信息只能支持中置信度判断：如果用户不恐水、能听懂安全指令、身体状态正常，并且潜店能提供充分讲解和小比例带客，可以考虑；否则建议选择浮潜或放弃下水活动。
